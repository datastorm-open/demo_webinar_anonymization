# webinar_anonymization

Code and results presented during DataStorm's webinar about anonymization.

## Dataset

- Generated randomly + "real" data (respondents was asked to lie) about employees in a company
- Directly identifying column (First name, name)
- Quasi-identifer (genre, birthdate, their place in the company, ...)
- Sensitive data (comorbidities)
